2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Compile contracts**
   ```bash
   npx hardhat compile
   ```

4. **Run tests**
   ```bash
   npx hardhat test
   ```

5. **Deploy contracts**
   ```bash
   npx hardhat run scripts/deploy_vision_2030_contracts.js --network <network>
   ```
