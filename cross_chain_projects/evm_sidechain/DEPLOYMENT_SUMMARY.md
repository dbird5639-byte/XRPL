## 🚀 Deployment Instructions

### Prerequisites
- Node.js 16+
- npm or yarn
- Hardhat
- Git

### Quick Start
```bash
# 1. Install dependencies
npm install

# 2. Compile contracts
npm run compile

# 3. Run tests
npm test

# 4. Deploy contracts
npm run deploy
```

### Network-Specific Deployment
```bash
# Local deployment
npm run deploy:local

# Testnet deployment
npm run deploy:testnet

# Mainnet deployment
npm run deploy:mainnet
```
